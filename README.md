# Configuration et Installation

Avant de lancer l'application, vous devez configurer votre environnement Python. 

### Installation des dépendances (PyTorch)

Utilisez la commande `pip` pour installer les versions spécifiques de PyTorch, TorchVision et TorchAudio :

```bash
pip install torch torchvision torchaudio --index-url [https://download.pytorch.org/whl/cu121](https://download.pytorch.org/whl/cu121)
